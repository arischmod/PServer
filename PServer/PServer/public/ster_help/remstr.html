-remstr
template: ster?clnt=<name|pass>&com=remstr[&str=str_pattern&str=str_pattern&...]
          Order of query params is not important.
pattern : A stereotype pattern can use * to match the last part of a stereotype name.
	  so "ster=some_ster.*" will return all stereotypes whose names begin with
	  some_ster. while "ster=*" will return results from all stereotypes.
descript: Removes all stereotypes defined by the str_patterns and all
	  information associated with them. Defining no str_pattern
	  will delete ALL stereotypes.
example : ster?clnt=<name|pass>&com=remstr&str=testStr&str=testStr2.*
	  ster?clnt=<name|pass>&com=remstr
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
