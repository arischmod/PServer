-addftr
template: ster?clnt=<name|pass>&com=addftr&str=str_name[&ftr1=dgr1&ftr2=dgr2....]
          Order of query params is not important. Stereotype and feature
	  names must not end in *. If no features are specified nothing
	  will be added.
descript: Adds all specified feature - degree pairs to the specified stereotype
example : ster?clnt=<name|pass>&com=addftr&str=testStr1&testFeature1=15&testftr2=8
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
