-addftr
template: ster?clnt=<name|pass>&com=addftr&str=<str>&ftr1=dgr1[&ftr2=dgr2...]
          Order of query params is not important.
descript: Adds to stereotype <str> all features specified using the corresponding
	  degrees. If a feature already exists in the stereotype it instead sets its
	  degree to the new degree specified.
example : ster?clnt=name|pass&com=addftr&str=test.1&testFtr.1=2&testFtr.5=0.1
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
200 OK  : example result body:
	  <result>
	    <row>
	      <num_of_rows>1</num_of_rows>
	    </row>
	  </result>
