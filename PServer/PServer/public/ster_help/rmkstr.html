-rmkstr
template: ster?clnt=<name|pass>&com=rmkstr&str=str_name
          Order of query params is not important. Stereotype name
	  must not end with *.
descript: Resets a stereotype by removing all its users and features
	  and adding users and their respective features based on the
	  stereotypes rule.
example : ster?clnt=<name|pass>&com=rmkstr&str=testStr
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
