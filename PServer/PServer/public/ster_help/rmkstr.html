-rmkstr
template: ster?clnt=<name|pass>&com=rmkstr&str=<str>
          Order of query params is not important.
descript: Removes and reinserts the sterotype specified removing all current
          users and features and adding again all users matching the stereotypes
          rule.
example : ster?clnt=name|pass&com=rmkstr&str=test.1
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
200 OK  : example result body:
	  <result>
	    <row>
	      <num_of_rows>1</num_of_rows>
	    </row>
	  </result>
